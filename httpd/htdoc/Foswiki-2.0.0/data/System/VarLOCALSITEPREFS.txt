---+ LOCALSITEPREFS -- web.topicname of site preferences topic
   * The full name of the local site preferences topic. These local site preferences overload the system level preferences defined in [[%SYSTEMWEB%.%WIKIPREFSTOPIC%]].
---++ Examples
   * =%<nop>LOCALSITEPREFS%= expands to =%LOCALSITEPREFS%=, renders as %LOCALSITEPREFS%
