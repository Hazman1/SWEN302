---+ SKIN -- current skin
=%<nop>SKIN%= expands the skin search path. For instance, =SKIN= can be set to =catskin, bearskin=.
The =SKIN= setting can be overridden using the URL parameter =skin=, such as =?skin=catskin,bearskin=
You can also extend the existing skin path using covers - see [[VarCOVER][COVER]]
---++ Examples
   * =%<nop>SKIN%= expands to =%SKIN%=
   * See [[Skins]] for more information
